Every time you want to deal with the value of a "variable" you write two code paths: one for Just (it has a value) and one for Nothing (it has no value). In JavaScript you do this with: if (req && req.body && req.body.data && req.body.data.name) {      // do something with req.body.data.name...   }   This is a programming convention, however, and it's not enforced by the compiler - so it can easily be left out.
